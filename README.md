## âš¡ Getting Started

### Prerequisites

- Node.js 18.17 or later
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Abdullahkhan1010/Portfolio.git

# Navigate to the project directory
cd Portfolio

# Install dependencies
```
